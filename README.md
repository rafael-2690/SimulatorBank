# SimulatorBank

Projeto simples em Java que simula operações básicas de um banco digital no terminal.

Foi desenvolvido como exercício prático, com foco em lógica de programação, estruturas de decisão e interação com o usuário pelo console.

## Funcionalidades

- Consultar saldo
- Receber valores
- Transferir valores
- Encerrar o programa

## Tecnologias

- Java
- IntelliJ IDEA

## Como executar

1. Clone este repositório:

```bash
git clone https://github.com/seu-usuario/SimulatorBank.git
```

2. Abra o projeto na sua IDE de preferência.

3. Execute a classe `Main.java`.

Se preferir rodar pelo terminal:

```bash
javac main/src/Main.java
java -cp main/src Main
```

## Estrutura do projeto

```text
SimulatorBank/
```

## Objetivo

Este projeto tem como objetivo praticar conceitos iniciais de Java, como:

- Variáveis
- Entrada de dados com `Scanner`
- Estruturas condicionais
- Laços de repetição
- Operações com saldo bancário
